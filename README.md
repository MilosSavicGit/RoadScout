# RoadScout

Discover the places worth stopping for.

## Version 1.0

### Features

-   Route exploration
-   Nearby (3 km) exploration
-   Multiple route alternatives
-   Historic sites, museums, heritage, viewpoints, public art and
    notable nature
-   Community voting
-   Community photo uploads (up to 5 photos)
-   Community gallery
-   Wikipedia descriptions
-   Translation
-   Android tested
-   iPhone Safari tested
-   GitHub Pages deployment

## Technology

-   MapLibre GL
-   Turf.js
-   OSRM
-   Overpass
-   Nominatim
-   Wikipedia
-   Supabase
