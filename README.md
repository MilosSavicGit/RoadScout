# RoadScout

**Part of ScoutPlatform**

RoadScout is a browser-based and Android-compatible geospatial
application for discovering historic sites, museums, heritage locations,
viewpoints, public art and notable natural attractions either **along a
driving route** or **within a 3 km radius** of the user's current
location.

RoadScout is one of the core production applications built on
**ScoutPlatform**, a family of lightweight exploration tools designed to
help people **discover, document and share knowledge about places**
using open geographic resources.

------------------------------------------------------------------------

# Overview

Unlike a conventional navigation application, RoadScout performs live
semantic searches around a selected route or around the user's current
location. It enriches results using multiple open data sources and
allows visitors to contribute photographs, local descriptions and
community ratings.

The application is designed to remain lightweight while providing
worldwide coverage using live geographic data.

# Design Philosophy

RoadScout intentionally avoids downloading large regional datasets.

Workflow:

1.  Select a route or current location
2.  Perform a live search
3.  Display worthwhile attractions in travel order

This keeps the application lightweight while always working with current
OpenStreetMap information.
